# E-Commerce Web Application

## Project Overview
This project is a responsive one-page E-Commerce web application built using React, TypeScript, and Tailwind CSS. The application displays products and allows users to add or remove items from the shopping cart.

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Vite


## Features
- Responsive single-page layout
- Header and Hero section
- Product listing (6 products)
- Product card with image, title, and price
- Add to Cart functionality
- Remove items from cart
- Dynamic cart item c
-
- ount
